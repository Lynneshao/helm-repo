📦 Quick Start

# 添加仓库 helm repo add myrepo https://kweaver-ai.github.io/helm-repo/ # 更新仓库索引 helm repo update # 搜索可用的 charts helm search repo myrepo # 安装 chart helm install demo myrepo/<chart-name> --version <x.y.z>
# Add repository helm repo add myrepo https://kweaver-ai.github.io/helm-repo/ # Update repository index helm repo update # Search available charts helm search repo myrepo # Install chart helm install demo myrepo/<chart-name> --version <x.y.z>
